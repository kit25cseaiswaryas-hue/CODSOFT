# Job Board Website

## CODSOFT Web Development Internship - Task 1

### Project Overview
The Job Board Website is a responsive web application that connects job seekers with employers. Users can browse jobs, apply for positions, and employers can manage job listings through a simple and user-friendly interface.

## Features
- Home Page
- Job Listings
- Job Details
- Candidate Dashboard
- Employer Dashboard
- Login Page
- Job Application Form
- Contact Page
- Blog Page
- Responsive Design

## Technologies Used
- HTML5
- CSS3
- JavaScript

## Project Structure


## How to Run
1. Download or clone the repository.
2. Open the project folder.
3. Open `index.html` in your web browser.
